  À FAIRE APRÈS OUVERTURE AU PUBLIC (retirer le noindex)
----------------------------------------------------------------

3. ACTIVER L'INDEXATION GOOGLE
   Quand le restaurant est prêt à être trouvé sur Google :
   a) Dans robots.txt : remplacer "Disallow: /" par "Allow: /"
   b) Supprimer la ligne <meta name="robots" content="noindex, nofollow">
      dans les 5 fichiers HTML (ou changer en "index, follow")
   c) Soumettre le sitemap dans Google Search Console

4. ANALYTICS (Google Analytics GA4)
   - Créer un compte sur analytics.google.com
   - Récupérer l'ID de mesure (format G-XXXXXXXXXX)
   - L'insérer dans les 5 pages HTML
   - Activer dans la bannière cookie (consentement avant activation)

5. DOMAINE www.kabanahouse.fr
   - Connecter le domaine au projet Vercel
   - Vérifier le certificat SSL actif
   - Tester la redirection HTTP → HTTPS

6. COMPRESSION IMAGES LEGACY (optionnel)
   JPG d'origine peuvent être optimisés :
   - salle-banquettes.jpg, bar.jpg, langouste.jpg
   - Outil : squoosh.app ou imageoptim
